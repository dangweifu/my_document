# 链接
Markdown 支持两种形式的链接语法：行内式和参考式两种
### 行内式：
\[链接标题]\(链接地址) [百度](https://baidu.com)

示例：\[百度](https://baidu.com)
### 参考式：
第一步：定义链接：\[链接变量名]: 链接地址
注意：冒号与链接地址之间有空格。
[baidu]: https://baidu.com
示例：\[baidu]: https://baidu.com

第二步：引用定义的链接，\[链接名称]\[链接变量名]

示例： [百度][baidu]
